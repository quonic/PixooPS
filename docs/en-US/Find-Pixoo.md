# Find-Pixoo

## SYNOPSIS

Finds the IP address of any Pixoo64 device on the local network

## SYNTAX

```powershell
Find-Pixoo [[-IPAddress] <Object>] [<CommonParameters>]
```

## DESCRIPTION

Finds the IP address of any Pixoo64 device on the local network

## EXAMPLES

### EXAMPLE 1

```powershell
Find-Pixoo
```
