Find-Pixoo

SYNOPSIS

Finds the IP address of any Pixoo64 device on the local network

SYNTAX

Find-Pixoo [[-IPAddress] <Object>] [<CommonParameters>]

DESCRIPTION

Finds the IP address of any Pixoo64 device on the local network

EXAMPLES

EXAMPLE 1

Find-Pixoo

PARAMETERS

-IPAddress

Specify only if you already know what the IP address is Can also be used to verify that the specified IP address is a Pixoo64 device
